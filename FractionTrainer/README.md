# FractionTrainer

Проект по теме 15: тренажер "собрать дробь".

## Возможности

- режим обучения с подсказкой;
- режим проверки знаний;
- кнопки `- доля` и `+ доля`;
- выбор секторов кликом мыши;
- счет за правильные ответы;
- переход к следующему примеру после правильного ответа;
- справка по `F1`;
- простой CLI-режим.

## Запуск

```bash
dotnet run
```

```bash
dotnet run --project FractionTrainer
```

CLI-пример:

```bash
dotnet run -- --task easy
```
