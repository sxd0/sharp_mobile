# WaterBalanceController

Свободная тема: контроллер водного баланса.

## Возможности

- дневная цель воды;
- быстрые кнопки добавления порций;
- добавление своей порции;
- прогресс за день;
- история сегодняшних записей;
- сохранение данных в пользовательский JSON-файл;
- справка по `F1`;
- простой CLI-режим.

## Запуск

```bash
dotnet run
```

CLI-примеры:

```bash
dotnet run -- --add 250
dotnet run -- --status
```
