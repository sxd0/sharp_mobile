# Проекты по мобильной разработке

Структура сделана по примеру прошлогоднего архива: два отдельных проекта в двух папках.

## Проекты

- `FractionTrainer` - тема 15, тренажер "собрать дробь".
- `WaterBalanceController` - свободная тема, контроллер водного баланса.

## Запуск

```bash
dotnet run --project FractionTrainer
dotnet run --project WaterBalanceController
```

## Сборка

```bash
dotnet build FractionTrainer
dotnet build WaterBalanceController
```
